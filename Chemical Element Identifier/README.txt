PROJECT TITLE: Chemical Element Identifier

DESCRIPTION:
This program aims to display a particular element with its name, chemical symbol, atomic number and atomic mass
and classify whether that certain element is an alkali metals, alkaline-earth metal, post-transition metals,
transition metal, lanthanide, actinide, metalloid, nonmetal, halogen or noble gas.

FILES:
elementIdentifier.cpp	- source code
table.h			- header file (supporting file for the source code)
elementIdentifier.exe	- executable file

DIRECTIONS:
- Download all the files included in this folder, and place them all in one folder.
- You can open the "elementIdentifier.exe" file only to view the program.
