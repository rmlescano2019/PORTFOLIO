PROJECT TITLE: RK Calculator

DESCRIPTION:
This program will be able to calculate the R (stadia constant) and K (stadia interval factor)
from a given data of a transit on a level ground having a horizontal sight and data in metric
and english units to be used in a topographic survey.

FILES:
RKCalculator.cpp	- source code
RKCalculator.exe	- executable file

DIRECTIONS:
- Download all the files included in this folder, and place them all in one folder.
- You can open the "elementIdentifier.exe" file only to view the program.
